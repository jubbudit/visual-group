What is this?
The Application

Visual Group is a visualization I made to graphically represent elements of cyclic groups as subgroups of the symmetric group. It was made using the JavaScript library p5.js. The controls and webpage formatting were created using the HTML Bootstrap library.

I had the idea to make this application while trying to explain to a friend the structural beauty of cyclic groups. I pulled out a pen and paper and tried to sketch out some permutations, but I found it difficult to draw polygons with more than 12 sides. Instead of learning how to draw, I decided to code up this nifty web app.

Note: there is a direction that the indexes permute. I haven't found a way to draw an arrow in p5 yet, so this is the best I can do for now.

What's a Cyclic Group?

Well... There's a lot to say about cyclic groups, but I'll keep it brief. A cyclic group is a group generated by a single element. This means that you apply an operation to the same element over and over again. In the case of this application, that means that the permutation you choose is applied to the identity element over and over again ("Power to Generator" number of times). Eventually, once you apply the permutation so many times (the order of the element number of times to be precise) the indexes will permute back to the identity. And then it starts all over again. Hence, it is a cyclic group.
